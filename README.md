easeljs-sokoban
===============

A Sokoban game written with the Easeljs canvas framework for a tutsplus video course.


### Install ###
Install node and npm.

Then;
```
npm install -g bower
npm install -g grunt
```
Then from the project folder
```
bower install
npm install
```

### Run ###
```
grunt
```

navigate to localhost:9001 in a modern browser
